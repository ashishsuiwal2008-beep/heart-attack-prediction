# ❤️ Heart Attack Prediction System

This project uses Machine Learning to predict the risk of heart attack based on patient health data.

## 📌 Features
- Uses Random Forest Classifier
- Predicts heart attack risk
- Simple and beginner-friendly

## 📊 Dataset
Dataset should include:
- age
- sex
- blood pressure
- cholesterol
- sugar level
- heart rate
- target (0 = no risk, 1 = risk)

## ⚙️ Installation

pip install -r requirements.txt
